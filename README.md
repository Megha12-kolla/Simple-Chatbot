# 🤖 Simple Python Chatbot

A beginner-friendly command-line chatbot built with Python that interacts with users through predefined responses. The chatbot can greet users, answer simple questions, tell a joke, and continue the conversation until the user decides to exit.

## 🌟 Features

* 👋 Greets the user by name.
* 💬 Responds to common greetings like **Hello** and **Hi**.
* 😊 Answers simple questions about itself.
* ❤️ Shares its favorite color.
* 😂 Tells a programming joke.
* ☀️ Responds to "Good morning".
* 🙏 Replies to "Thank you".
* 👋 Ends the conversation when the user types **bye**.
* 🔄 Keeps chatting until the user chooses to exit.

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Open a terminal in the project folder.
3. Run the program using:

```bash
python chatbot.py
```

## 💻 Example

```text
Hello! What's your name? Maggie
Nice to meet you, Maggie

You: hello
Bot: Hi there!

You: tell me a joke
Bot: Why did the programmer quit his job? Because he didn't get arrays! 😄

You: bye
Bot: Goodbye! See you later!! 👋
```
